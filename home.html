KODI Add-ons Repository

Chào mừng đến với repository add-ons KODI của chúng tôi!

Tải Repository Add-on Tải HDVN Add-on

Hướng dẫn cài đặt:

  1. Tải xuống file repository add-on (repo_vietmediaf_offical-0.0.1.zip)
  2. Trong KODI, vào Settings > Add-ons > Install from zip file
  3. Chọn file repository add-on đã tải xuống
  4. Sau khi cài đặt repository, bạn có thể cài đặt các add-ons khác từ repository này
